Code

  • Sometimes VScode intelliSense by OmniSharp breaks, and doesn't show the hover of a HTML snippet

    Double check this setting in VScode:

    "editor.hover.enabled": true,

  • SCSS only needs to declare the class name ONCE. Try to nest stuff.

  • All pushed branches are shared, but only master branch is deployed!

  • Server Chasing for any files you need to make the user download again.
    ?v=16042022

    Example:
    "/css/views/service.css?v=310522"

  • flex basis
    calc(100% - 100px)

  • Bootstrap needs a .row class on an element to tell the element that .col classes and other bootstrap classes can be applied to it.

    .row > .span > .col .classes
